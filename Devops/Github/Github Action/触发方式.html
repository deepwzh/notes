触发方式

概述

流水线支持的触发方式：

  • push
  • 定时触发
  • 通过回调触发
  • 手动触发

示例

name: 部署文档

on:
  # 手动触发
  workflow_dispatch: {}
  # push的时触发 
  push:
    branches:
      - main
  # 定时触发
  schedule:
    - cron: '0 0 * * *'  # 每天零点运行一次（UTC+0)
  # webhook触发
  repository_dispatch:
      types: [feishu]
