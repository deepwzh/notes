常用工具

s3fs-fuse 磁盘挂载

s3fs 用于将对象存储挂载到磁盘上

https://github.com/s3fs-fuse/s3fs-fuse

首先安装工具

sudo apt-get install s3fs

使用这个命令添加配置

echo "key:secert" > ~/.passwd-s3fs
chmod 600 ~/.passwd-s3fs

启动挂载(临时)，如果要持久化，就需要修改/etc/fstab 文件了

s3fs tool /s3/tool -o passwd_file=${HOME}/.passwd-s3fs -o url=https://minio.example.org/ -o use_path_request_style -o dbglevel=info

可以使用 umount 直接取消挂载

umount /s3/tool
